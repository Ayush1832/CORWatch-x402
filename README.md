# CORWatch x402

**CORWatch x402** is a paid observability and validation surface for Cortensor.

## Structure

- `frontend/`: React/Vite Frontend
- `backend/`: Node.js/Express Backend (x402 enabled)
- `contracts/`: Solidity Smart Contracts
- `docs/`: Documentation

## Quick Start

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

See `docs/` for more details.
